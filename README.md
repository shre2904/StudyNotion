# Studynotion

Studynotion is an ed-tech platform built with React and Tailwind CSS.

## Usage

To run the project locally:

1. Install dependencies.

    ```sh
    npm install
    ```

1. Start the client.

    ```sh
    npm start
    ```

1. Start the server (separate terminal).

    ```sh
    npm run server
    ```

1. Start both together.

    ```sh
    npm run dev
    ```
1. Open the project in your browser at [`http://localhost:3000`](http://localhost:3000) to view your project.
1. Create your React components and add your styles using Tailwind classes. You can also create new CSS files and import them into your components.

Update backend branding variables in `server/.env`:

- `APP_NAME=Studynotion`
- `PUBLIC_APP_URL=http://localhost:3000` (set this to your deployed URL later)
- `SUPPORT_EMAIL=` (set this when you create one)
